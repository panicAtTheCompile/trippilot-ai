# TripPilot AI

TripPilot AI is a MERN-based travel assistant that allows users to upload travel booking information, generate itineraries, store trip history, and share itineraries through unique links.

## Features

* JWT Authentication (Register/Login)
* Travel document upload workflow
* Automated travel information extraction pipeline
* Itinerary generation service
* MongoDB trip storage
* Trip history dashboard
* Shareable itinerary links
* Responsive React frontend
* Express + MongoDB backend

## Tech Stack

Frontend:

* React
* Axios
* CSS

Backend:

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication

Deployment:

* Vercel (Frontend)
* Render (Backend)
